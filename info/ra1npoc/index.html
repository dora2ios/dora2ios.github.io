checkra1n dump and poc for iphoneos

This is the poc code for running checkra1n on iphoneos, based on the payload dumped from checkra1n 0.12.4 beta.

All at your own risk. Do not run it on a normal device.

Usage: ra1npoc [option]

-h: show usage

-l: show list of supported devices

-v: enable verbose boot

-c: use cleandfu [BETA]

-d: enable debug log

-e <str>: set extra bootargs

-s: use special pongo_2.5.0-0cb6126f

Note1: In order to use this package with lightning devices, you need to use lightning to USB camera adapter.

Note2: Using this package with lightning devices, depending on the battery level of the device, there may be a shortage of power supply. In this case, please use a self-powered USB hub connected to the lightning to USB camera adapter.

Note3: If you want to boot A9X, you must add '-s' flag.
