This is the untethered exploit for 32-bit installed by the daibutsu 8.4.1 jailbreak tool. If you jailbroke your 32-bit device with daibutsu 8.4.1, this untether has already been automatically installed for you.

Note1: This package and jailbreak was created to learn and demonstrate the old jailbreak style that was used about 6 years ago. All of these are provided for educational and learning purposes, and the author and provider assume no responsibility for them. Abusing these is never acceptable, so never do it!

We strongly recommend that you update your device's iOS to the latest version to improve the security of your device.

Note2: Upgrading this package will require a reboot. Since iOS 8, if you upgrade Cydia at the same time as installing a package that asks for a reboot, Cydia might get killed, and you might fail to be asked to perform the reboot. We recommend upgrading manually if this happens.

v1.1: This version of the package, vm_fault_enter patch has been added to kernel patch and improve the reliability of jailbreak.

v1.2: This version of the package now supports TaiG style AFC2 package. This makes it possible to enable full file access without relying on Mobile Substrate.

v1.2.1: This version of the package changes the loading format of AFC2 so that executables under /etc/rc.d are executed at the time they are loaded by untether.

v1.2.2: This version of the package has a smaller size of the untether executable binary. This version is the first official release version.

v1.2.3: This version of the package fixes a mistake in permissions for some files during jailbreak installation. It has also been modified to make it behave properly when injecting daibutsu in the CFW base.
